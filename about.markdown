Iket is an API gateway focused on secure operations, extensibility, and practical day-two workflows. It is built in Go and designed for teams that want a gateway they can run, inspect, and evolve without giving up modern platform controls.

## What makes it different

<div class="feature-grid">
  <div class="feature-card">
    <h3>Remote administration with mTLS</h3>
    <p>Operational access stays locked down without giving up day-to-day usability.</p>
  </div>
  <div class="feature-card">
    <h3>Proposal, queue, and canary workflows</h3>
    <p>Config changes become safer in shared environments with better visibility and control.</p>
  </div>
  <div class="feature-card">
    <h3>Pluggable architecture</h3>
    <p>Custom middleware and lifecycle integrations fit in cleanly without reinventing the core gateway.</p>
  </div>
  <div class="feature-card">
    <h3>Protocol-aware routing and policy controls</h3>
    <p>Teams can run classic APIs and agent-facing traffic from the same operational surface.</p>
  </div>
</div>
