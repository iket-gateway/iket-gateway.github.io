Iket is an API gateway focused on secure operations, extensibility, and practical day-two workflows. It is built in Go and designed for teams that want a gateway they can run, inspect, and evolve without giving up modern platform controls.

## Project Links

- [GitHub repository](https://github.com/bhangun/iket)
- [Releases](https://github.com/bhangun/iket/releases)
- [Issue tracker](https://github.com/bhangun/iket/issues)
- [Documentation hub](/docs/)

## What makes it different

- **Remote administration with mTLS** keeps operational access locked down without giving up usability.
- **Proposal, queue, and canary workflows** make config changes safer in shared environments.
- **Pluggable architecture** allows custom middleware and lifecycle integrations without reinventing the whole gateway.
- **Protocol-aware routing and policy controls** help teams run both classic APIs and agent or AI-facing traffic from the same control plane.
